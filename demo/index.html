How to use

To run the demonstration on the 3 list types supported by dd-poly, duplicate listSpecs-links.js, listSpecs-single.js, or listSpecs-multi.js in the dd-poly directory and change the name of the duplicate to listSpecs.js, replacing the existing file of that name. You may also amend or replace the contents of the file if you maintain the format.

The handling of pointer (mouse) navigation in this version is incomplete, so some pointer actions can create keyboard traps.

Activating the commit button with the single and multi list types does nothing except disable the button until a selection change occurs.
